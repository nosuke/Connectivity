Maquette du projet de IF10 Connectivity.

 

Terminaux cibles : ordinateurs. Nous avons décidé de cibler ce type de terminal car bien que les terminaux mobiles gagnent énormément d'ampleur au cours de ces dernières années, les sites web accédés via un ordinateur restent majoritaires. De plus, notre groupe a de meilleures connaissances en web et en design sur terminaux standards plutôt qu'en terminaux mobiles. Mais rien n'empêche dans un second temps de tourner notre service vers un déploiement sur smartphone. La maquette a été réalisée avec une largeur fixe, mais le site se veut responsive design dans la réalité.

 

Connexion

Permet de s'authentifier sur le site.

• Le couple identifiant/mot de passe est root/root.

• On peut afficher et masquer le mot de passe grâce au bouton à droite du champ "mot de passe".

• Les deux cases à cocher en dessous permettent de se souvenir de l'identifiant et/ou du mot de passe. Si la case "se souvenir de mon mot de passe" est cochée, un message d'avertissement apparaît pour avertir l'utilisateur du danger de cette option.

• Un message d'alerte apparaît également si le couple identifiant/mot de passe est incorrect.

• Une fois la connexion réussie, l'utilisateur est redirigé vers la page d'accueil.

 

Inscription

Permet de créer son compte.

• Un message d'erreur apparaît à droite des champs obligatoires si ces derniers ne sont pas renseignés.

• Le champ "confirmation du mot de passe" doit correspondre au champ "mot de passe" afin d'éviter que l'utilisateur se trompe lors de la saisie de son mot de passe.

• Un champ d'information apparaît à droite du champ "mot de passe" afin de renseigner l'utilisateur quant aux spécificités du mot de passe (ex : au moins 8 caractères).

 

Accueil

Page principal du site

Comporte 3 onglets : accueil (la carte), forums et à propos.

 

Carte

La majorité des interactions de la maquette sont présentes sur cette page. Sur cette maquette la carte est une simple image. En réalité, elle serait une carte interactive de type google map.

Le menu à gauche de la carte (qui peut être masqué) permet de :

• Ajouter un marqueur (point d'accès wifi) sur la carte. Une fenêtre s'ouvre alors pour renseigner les différentes informations concernant le point d'accès, avec vérification de la cohérence des données.

• Naviguer sur la carte via une adresse. Cette action est simulée ici par la transition entre deux images. On peut ainsi renseigné "utt" ou "troyes".

• Filtrer les points d'accès wifi présents sur la carte. En sélectionnant certains critères, les marqueurs sur la carte qui ne correspondent pas sont masqués. Le critère "fournisseur" est ici à titre illustratif et n'intervient pas dans le filtrage.

 

En cliquant sur un marqueur sur la carte, une fenêtre s'ouvre. On peut :

• Ajouter le marqueur à ses favoris (le marqueur devient alors jaune). Pour réaliser cette action, il faut être connecté. Si ce n'est pas le cas, on est redirigé vers la page de connexion.

• Laisser un commentaire ou lire ceux des autres utilisateurs. En cliquant sur ce lien, on arrive sur la page des commentaires.

• Noter le point d'accès en cliquant sur une étoile.

 

Forums

Cette page représente un forum. Aucune interaction particulière.

 

À propos

Cette page donne des informations sur Connectivity. Aucune interaction particulière.

 

Commentaires

On peut sur cette page lire les commentaires des autres utilisateurs concernant une borne wifi particulière, ou rédiger notre avis. Il suffit alors de remplir la zone de texte et de valider. On doit être authentifier pour réaliser cette action.

Dans cette maquette, cette page est identique peu importe le point d'accès qu'on souhaite commenter. En réalité elle s'adaptera bien entendu au point sur lequel on a cliqué. De plus, cette maquette ne fait que simuler l'ajout d'un commentaire puisqu'en effet, si on tente d'ajouter un second commentaire, le premier sera remplacé.
